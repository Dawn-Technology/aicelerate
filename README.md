# aimate — AI Automation Teammate by Dawn Technology

A GitHub Copilot CLI plugin marketplace by Dawn Technology, containing the `aimate` plugin with reusable skills for security auditing, GitLab workflow automation, and development planning.

## Install

### Via the marketplace (recommended)

Register the marketplace once, then install by name:

```bash
copilot plugin marketplace add Dawn-Technology/aimate
copilot plugin install aimate@aicelerate
```

### Direct install

```bash
copilot plugin install Dawn-Technology/aimate:plugins/aimate
```

### Update

```bash
copilot plugin update aimate
```

## Requirements

- GitHub Dawn Technology organization account with Copilot seat enabled
- [GitHub Copilot CLI](https://docs.github.com/en/copilot/how-tos/copilot-cli) installed and authenticated
